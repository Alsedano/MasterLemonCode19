# Readme para ejercicio básico

He creado dos componentes de _Lista_ y de _Detalle_, uno para los miembros de Github y otro para el de Rick y Morty.

En un principio hize uno común para los 2, pero al estilarlo con Material-UI he preferido separarlos porque en uno uso un tipo de paginación en el grid diferente. El de Github se autocachea solo y pide a la api los miembros de una forma que no necesita el useEffect. Por eso he preferido separarlos y usar el useEffect en uno de ellos.

También he decidido combinar el uso de estilado de Material-UI y de CSS, SCSS en las aplicaciones de los ejercicios para practicar la primera parte de Layout que vimos con Fran.
